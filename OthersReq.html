5. Other Nonfunctional Requirements

5.1 Performance Requirements

TO-LET (our project) must be active and responsive 24/7. Any request must be processed within 3 seconds. Our site should be loaded or browsed or refreshed within 2 seconds. Any sort of error must be handled within 2 seconds. For opening, sorting, computing, posting of rentable or sellable house lists or any other files must be done within 2 seconds.

5.2 Safety Requirements

If there is any extensive damage to a wide portion of the database due to catastrophic failure, such as a disk crash, all the user information and uploaded files are saved in the archived database for a backup and the recovery method restores a past copy of the database for safety purpose.

5.3 Security Requirements
