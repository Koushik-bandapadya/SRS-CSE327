TO-LET(BashaBharaChai)

There are many website/apps in bangladesh for solving renting home solutions. Our TO-LET is also like this. So, there may be rise question, why we need another website like this "TO-LET". Tolet is a rentable house finding solution, which is differ with other sites like Bikoy/ bdproperty /bproperty etc. Because In TO-LET, user can search rentable houses through mobile devices with the facility of VR and AR. User can open TO-LET app, click camera view and get real time data from the houses in front of the user. It helps the user to find any rentable house near him/her and makes User life easier . It uses the data of renowned sites like Bikroy, bdproperty etc by web scrapping. So users can find all houses detail in one place.
